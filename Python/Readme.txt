In this Python project program, we have created a quiz game.

In this game user is supposed to answer 10 question for the quiz game.

Based upon which the user will be given marks for each correct question.

If at all user does not want to play he can type any key and exit the game.

The basic score as user start playing will be initially 0.

All the marks will be calculated at the end of the quiz and will be displayed to the user.

The questions are set upon basic and Modern computer technalities.

Upon each correct answer given by user, the user will get to know the anser if it is correct or not.
